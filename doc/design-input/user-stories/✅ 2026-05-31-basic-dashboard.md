# User Story: Basic Dashboard

As a logged in user into Nodl, tho wants to use the app to record and transcribe new audio, 
I want to have a basic dashboard, where I can 
  - see my already recorded sessions,
  - create a new recording session,
  - see my transformers and the default transformer,
  - see my finished documents,
So that I have a quick and cleanly organized way to start using the application.

## Acceptance Criteria

- The user interface shall be in english.
