# User Story: Always see integrity Status

As a logged in user, I want to always see the status of the integrity check, even if it is turned off for me, so that I know this feature exists and could be a useful feature for me I wanna use in the future.

## Acceptance Criteria

- Users who don't have the integrity check (tamper protection) feature turned off, can also see the status element on the top above the transcript.
- It shows a gray symbol and tells the user this feature is currently turned off.
- When expanding, the user sees a message reading "Nodl can provide a certificate that proves the audio recording was not altered and existed before. Verifyiable by official bodies"
