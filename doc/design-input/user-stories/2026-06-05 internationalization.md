# User Story: Internationalization 

As a user (logged in or not), I want to use and experience the app in my native language, so that I have a good user experience.

## Acceptance Criteria

- Supported languages are English (current language), and German (new translation).
- A simple language switcher on the landing page lets the user select between the supported languages (English and German for now).
- A logged in user can switch the language from the user drop down menu on the top right of the page.
- No nation flags are used to represent languages. Languages are languages, and nations are nations.
- i18n standard pratices and i18n best pratices are respected and followed.
- A new AI-Agent skill exists with supporting scripts, that allow for automatic translation of missing parts (delta).
- The app is english first, with German translation in clean and complete translation files.
- For the German translation: Typical anglizismen are kept in the app.
- The app is fully and completely translated into German.
