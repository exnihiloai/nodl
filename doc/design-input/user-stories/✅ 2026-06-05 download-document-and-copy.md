# User Story: Download Document and Copy

As a user with transcribed audio recordings and documents, 
I want to be able to download the finished document, and additionally copy the text including text formatting and styling,
so that I can paste it into my other programs, or continue working with the downloaded file.

## Acceptance Criteria

- On the document view page, in the container that holds the document text, there is a copy button in the top right corner.
  - When pressing this button, the entire content of the document is copied into the clipboard, including text formatting and styling.
- The document can be downloaded via a download button on the top in the following formats:
  - PDF
  - MS Word
  - Markdown
