# User Story: See Password in Password Field

As a user, who is using a phone, or often makes spelling errors, I want to be able to see what I have typed on all fields where I need to enter my password (e.g. during sign-up and login), so that I can check what I typed and fixe mistakes without guessing.

## Acceptance Critera

- On each password field, a button allows the user to see their password.
- No password is ever stored in plain text.
