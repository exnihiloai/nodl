# User Story: Uphold 3rd License Obligations

As a maintainer of this application, I want to uphold all obligations imposed by licenses of 3rd party software used within this application, so that I respect copyright law and comply with regulation.

## Acceptance Criteria

- All 3rd party software is identified and documented in an OTS design output document.
  - This includes not only libraries and packages, but also fonts, or icons (lucide) and similar assets.
  - We do not consider the operating system, its parts, or the kernel to be part of the application.
- For each 3rd party software the license is identified.
- A dedicated page, accessible from the about page lists all 3rd party software that **require to be mentioned** by their license.
  - This page also reproduces a copy of their respective license texts if those licenses require to do so.
  - The pages https://zedl.app/about and https://zedl.app/licenses are taken as reference for this page or pages subject of this user story.
