# User Story: Push Notifications

As a user who wants to speak into their diary every day, I want to receive a push notification from Nodl, so that I am reminded of my daily habit to use Nodl.

## Acceptance Criteria

- A new settings page is available from the drop down menu. 
  - The menu entriy "Setting" sits below the language switcher and above the "Upgrade" and "Logout" entries.
  - Clicking on the "Settings" entry brings the user to a new "Settings" page.
- The Settings page has a section for setting a daily reminder.
- A user can enable or disable the daily reminder using a toggle switch
- A user can set the notification time using a simple time picker with HH:MM (24 hour format)
- A user can chose the Notification text with up to 30 chars. 
  - Default german : "Heute schon genodelt?", english: "Already nodeled today?"
- When the option is enabled, a user will receive a push notification on their mobile device.
